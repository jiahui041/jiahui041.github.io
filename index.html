Projects

1. Hidden Quest – Social Computing Platform

View Report

Hidden Quest is a gamified social computing platform designed to encourage authentic, meaningful digital interactions. I lead the UX process from problem discovery to interactive prototyping by conducting user interviews, mapping emotional user journeys, and running iterative usability tests. Using Figma, I crafted an interface inspired by user empathy, accessibility principles, and game mechanics to create a seamless, engaging experience.

  • Skills: UX Research, Empathy Mapping, Gamification, Interaction Design, Figma

2. UCSD WebReg Redesign

This project reinterpret UC San Diego's WebReg course registration system. I collaborated with a team to uncover student frustrations and pain points, then led the design of a modernized interface focused on clarity, flexibility, and visual scheduling. By integrating usability heuristics and accessibility standards, we built a responsive, intuitive prototype that improved the student experience during high-stress enrollment periods.

  • Skills: UX/UI Design, Information Architecture, Accessibility, Team Collaboration, Prototyping

3. WellTrack Prototype

WellTrack is a wellness and mental health tracking app designed for college students. I led the design and prototyping process, creating an experience that supports emotional awareness and habit-building. Through research and iterative feedback cycles, I designed onboarding flows, interactive progress charts, and accessible UI components to support mental health journeys in a non-judgmental and user-friendly way.

  • Skills: High-Fidelity Prototyping, UX Writing, UI Animation, Accessibility Design, Emotional Mapping

4. Clean Air Challenge – CAPS Committee

Clean Air Challenge Report

The Clean Air Challenge is an educational simulation game developed in collaboration with the UCSD Design Lab to raise awareness about air pollution and its impact. Built using Construct 3 through our high-fidelity prototype, the game leverages interactive storytelling, real-time feedback, and decision-making mechanics to immerse users in scenarios where their choices influence air quality outcomes. I led the game design and UX strategy, ensuring the experience was both engaging and informative, especially for younger audiences and community outreach events.

  • Skills: Game Design, Construct 3, UX Strategy, Interactive Storytelling, Environmental Education

5. Eaglesoft UX Redesign

View Project

Redesigned Patterson Eaglesoft’s legacy dental software to address usability concerns and optimize user workflows. I worked closely with users and stakeholders to identify interface inefficiencies, then led the redesign by mapping tasks, simplifying navigation, and introducing modern visual systems. The final prototype introduced an x-ray comparison module and integrated patient messaging, significantly improving operational clarity for dental professionals.

  • Skills: Workflow Design, UI Modernization, Heuristic Evaluation, Stakeholder Communication, Design Systems
